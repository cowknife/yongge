### CF vless代码默认修改内容

1、UUID默认已更换，建议自定义

2、proxyIP默认为香港地区的固定IP，可自定义

3、伪装网页默认为央视海外频道global.cctv.com，可自定义

4、对配置信息结果作了workers与pages优化显示
